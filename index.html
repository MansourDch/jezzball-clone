JezzBall Clone

Level: 1 Lives: 3 Filled: 0%

Controls

  • Spacebar: Start game / Draw line
  • Left Arrow or A: Move paddle left
  • Right Arrow or D: Move paddle right
  • Button Click: Start Line (alternative to spacebar)

Objective: Keep the ball bouncing with the paddle and fill the screen while clearing enemies. Each life lost when the ball falls below the paddle.
